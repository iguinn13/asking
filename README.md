<h1>Asking</h1>
<p>Asking é uma plataforma de perguntas e respostas com o formato anônimo. Criei a plataforma para desenvolver meus conhecimentos em back-end.</p>
<h2>Tecnologias utilizadas:</h2>
<ul>
  <li>NodeJS</li>
  <li>Express</li>
  <li>BodyParser</li>
  <li>Sequelize</li>
  <li>EJS</li>
</ul>
